关于本站

GitHub 提供了一种功能，叫 GitHub Pages, 利用这个功能，我们可以为项目建立网站。Jekyll是一个简单的，针对博客设计的静态网站生成器。使用 GitHub 和 Jekyll，我们可以打造自己的独立博客。我的网站使用了Bootstrap的一些基础模块。利用jQuery插件实现部分功能。一些比较炫的效果是Canvas 生成的。而关于代码高亮google-code-prettify/Gist都可以实现。
